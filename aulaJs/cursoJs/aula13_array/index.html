Array unidimensional: estaremos apreendendo o conceito de array.

Vetor unidimensional possuí apenas 1 indice.

declaração de array, usamos [] ou new Array()

Método push() inseri elemento no final do array.

método unshift() inseri no inicio do array.

método pop() remove do final do array.

método splice() remove de qualquer posição do array.

Método indexOf() pesquisar elemento dentro do array.

Método sort() ordenar o array em ordem crescente.

Método reverse() inverter ordens dos elementos do array

Método join() ele junta os elementos do array em uma variável só em forma de string, separados por virgula.

método concat() unificar dois arrays.

sempre primeiro posição é 0.
