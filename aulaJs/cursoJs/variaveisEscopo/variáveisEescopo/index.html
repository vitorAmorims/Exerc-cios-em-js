Aqui estamos, apreendendo o conceito de variáveis.

basicamente, variável: serve para armazenar valor em memória RAM.

Javascript não é fortemente típado, não preciso declarar o tipo de variável: string, number, boolean, array, real, float..., e podemos concatenar valores diferentes.

Importante, saber escopo para saber o limite das variáveis.

Para declarar variáveis em javascript usamos a palavra reservada var

Também, podemos usar a palavra reservada let para criação de variáveis.

O let é muito interessante, porque ele delimita o escopo de utilização da variável.

Podemos utilizar a palavra reservada const, para definir uma constante.

Importante lembrar das regras de criação de varáveis.

      não pode ter espaços.
      Não pode haver acentos
      não pode ter caracteres especiais.
      não deve iniciar com números.
      deve ser nome relacionado ao produto em questão.
      Podemos usar declaração implincita de variável, sem usar o comando var Exemplo: numero = 3;
